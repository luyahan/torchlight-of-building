# TLI Build Planner

Torchlight Infinite character build planner.

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- TanStack Start + Vite
- React 19
- TypeScript
- Tailwind CSS 4
- Zustand (state management)
